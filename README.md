# Soil Analysis & Micronutrient Classification for IoT Farms

This project analyzes soil nutrients and recommends crops using machine learning and IoT data.

## Features
- Soil nutrient analysis
- IoT sensor monitoring
- Crop prediction
- Micronutrient classification

## Technologies Used
- Python
- Machine Learning
- IoT Simulation
- Pandas
- Scikit-learn
- Matplotlib
